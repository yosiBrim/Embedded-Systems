# Lab 3: Motors Control and Interfacing

This folder contains the source code and documentation for the third laboratory. The project focuses on integrating and controlling motors using MicroPython on an ESP32 microcontroller.

## 🛠️ Tasks
1. **`task1_servo.py`:** Controls a Servo motor's angle via a 50Hz PWM signal mapped from an analog potentiometer input.
2. **`task2_stepper.py`:** Drives a Stepper motor using an A4988 driver. Implements a non-blocking state machine (using `time.ticks_ms()`) to respond to user buttons (direction and step count) while executing timed movements.
